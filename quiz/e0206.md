# Reverse Linked List

Reverse a singly linked list.

Example:

- Input: 1->2->3->4->5->NULL
- Output: 5->4->3->2->1->NULL

Follow up:

A linked list can be reversed either iteratively or recursively. Could you
implement both?

## Visualized Solution

```
0. Original Linked List:

    +===========+      +===========+                 +===========+
    |   head    |  +-->|           |  +-->       +-->|           |  +--> NULL
    +-----------+  |   +-----------+  |          |   +-----------+  |
    | value = 1 |  |   | value = 2 |  |    ...   |   | value = L |  |
    +-----------+  |   +-----------+  |          |   +-----------+  |
    |     p     |--+   |     p     |--+        --+   |     p     |--+
    +-----------+      +-----------+                 +-----------+

1. Initialize Pointers
   - prev := NULL
   - curr := head
   - next := undefined; temporarily store the next node before changing the curr's
     pointer.

    +===========+  
    |prev = NULL|
    +-----------+
    
    +===========+      +===========+                 +===========+
    |   curr    |  +-->|           |  +-->       +-->|           |  +--> NULL
    +-----------+  |   +-----------+  |          |   +-----------+  |
    | value = 1 |  |   | value = 2 |  |    ...   |   | value = L |  |
    +-----------+  |   +-----------+  |          |   +-----------+  |
    |     p     |--+   |     p     |--+        --+   |     p     |--+
    +-----------+      +-----------+                 +-----------+

2. Loop until `curr` is `NULL`:

    2.1.1. `next := curr.p`

    +===========+  
    |prev = NULL|
    +===========+
    
    +===========+      +===========+                 +===========+
    |   curr    |  +-->|   next    |  +-->       +-->|           |  +--> NULL
    +-----------+  |   +-----------+  |          |   +-----------+  |
    | value = 1 |  |   | value = 2 |  |    ...   |   | value = L |  |
    +-----------+  |   +-----------+  |          |   +-----------+  |
    |     p     |--+   |     p     |--+        --+   |     p     |--+
    +-----------+      +-----------+                 +-----------+

    2.1.2. `curr.p := prev`

    +===========+      +===========+
    |   curr    |  +-->|prev = NULL|
    +-----------+  |   +===========+
    | value = 1 |  | 
    +-----------+  |  
    |     p     |--+
    +-----------+    

    +===========+      +===========+                 +===========+
    |   next    |  +-->|           |  +-->       +-->|           |  +--> NULL
    +-----------+  |   +-----------+  |          |   +-----------+  |
    | value = 2 |  |   | value = 3 |  |    ...   |   | value = L |  |
    +-----------+  |   +-----------+  |          |   +-----------+  |
    |     p     |--+   |     p     |--+        --+   |     p     |--+
    +-----------+      +-----------+                 +-----------+

    2.1.3. `prev := curr`

    +===========+
    |   prev    |  +--> NULL
    +-----------+  |
    | value = 1 |  | 
    +-----------+  |  
    |     p     |--+
    +-----------+    

    +===========+      +===========+                 +===========+
    |   next    |  +-->|           |  +-->       +-->|           |  +--> NULL
    +-----------+  |   +-----------+  |          |   +-----------+  |
    | value = 2 |  |   | value = 3 |  |    ...   |   | value = L |  |
    +-----------+  |   +-----------+  |          |   +-----------+  |
    |     p     |--+   |     p     |--+        --+   |     p     |--+
    +-----------+      +-----------+                 +-----------+

    2.1.4. `curr, next := next, next.p`

    +===========+
    |   prev    |  +--> NULL
    +-----------+  |
    | value = 1 |  | 
    +-----------+  |  
    |     p     |--+
    +-----------+    

    +===========+      +===========+                 +===========+
    |   curr    |  +-->|   next    |  +-->       +-->|           |  +--> NULL
    +-----------+  |   +-----------+  |          |   +-----------+  |
    | value = 2 |  |   | value = 3 |  |    ...   |   | value = L |  |
    +-----------+  |   +-----------+  |          |   +-----------+  |
    |     p     |--+   |     p     |--+        --+   |     p     |--+
    +-----------+      +-----------+                 +-----------+

    2.2.

    +===========+      +===========+
    |   prev    |  +-->|           |  +--> NULL
    +-----------+  |   +-----------+  |
    | value = 2 |  |   | value = 1 |  | 
    +-----------+  |   +-----------+  |  
    |     p     |--+   |     p     |--+
    +-----------+      +-----------+    

    +===========+      +===========+                 +===========+
    |   curr    |  +-->|   next    |  +-->       +-->|           |  +--> NULL
    +-----------+  |   +-----------+  |          |   +-----------+  |
    | value = 3 |  |   | value = 4 |  |    ...   |   | value = L |  |
    +-----------+  |   +-----------+  |          |   +-----------+  |
    |     p     |--+   |     p     |--+        --+   |     p     |--+
    +-----------+      +-----------+                 +-----------+

    ...

    2.L

    +===========+      +===========+                 +===========+
    |   prev    |  +-->|           |  +-->       +-->|           |  +--> NULL
    +-----------+  |   +-----------+  |          |   +-----------+  |
    | value = L |  |   |value = L-1|  |    ...   |   | value = 1 |  |
    +-----------+  |   +-----------+  |          |   +-----------+  |
    |     p     |--+   |     p     |--+        --+   |     p     |--+
    +-----------+      +-----------+                 +-----------+

    +===========+  
    |curr = NULL|
    +===========+

3. Return `prev` as the new head.
```
